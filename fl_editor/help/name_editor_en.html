Name Editor

The Name Editor lets you manage ids_name entries centrally in resource DLLs (user DLL), without CSV workflows.

Features

  • Search: Filter by Global ID, name, or DLL.
  • Edit: Select existing names and save changes directly.
  • Create: Add new name entries (new Global ID).
  • Usage view: Clicking an ID shows all objects/zones and files using that ID.

Assign missing ids_name directly

  • The lower panel lists all objects/zones with ids_name = 0.
  • Enter a name and assign directly: FL Atlas creates the DLL entry and writes the new ID into the INI file.
  • No CSV files required.

Important

  • Changes are written to the configured user DLL, not vanilla DLLs.
  • Existing IDs can be reused and updated.
