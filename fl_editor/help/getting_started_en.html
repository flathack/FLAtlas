Getting Started

  1. Start FL Atlas and open the Welcome Screen (automatically on first start, or later via Help → Welcome Screen).
  2. Click Open Mod Manager.
  3. Set your Mod Repository and optional Clean FL test folder.
  4. Create a mod or select an existing entry, then click Open for Editing.
  5. The left system browser will show all detected star systems. Click a system to load it.
  6. Use 🌐 Universe to open the universe map with all systems.
  7. The active editing mod is shown in the top menu bar.

See also User Interface and Editing & Creation for more details.
