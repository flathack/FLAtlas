Getting Started

  1. Start FL Atlas and enter the game path at the top left (e.g. C:\Games\Freelancer).
  2. Click Read or press Enter.
  3. The left panel will show all detected star systems. Click a system to load it.
  4. Use 🌐 Universe to open the universe map with all systems.
  5. The last used paths and settings are saved automatically.

See also User Interface and Editing & Creation for more details.
