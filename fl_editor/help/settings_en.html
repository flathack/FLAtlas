Settings & Themes

Language & Theme

  • Language can be switched at any time (DE/EN button in the toolbar).
  • Theme selection: Founder, Dark, Light, Custom (your own accent color).
  • All settings are saved and restored on next start.
  • Default language for new users is English.

Global Settings

  • Global settings include language, theme, auto-name language, BINI conversion, and the freelancer.ini editor.
  • Path and mode management moved to Mod Manager (no duplicated path fields).
  • The active editing context is set in Mod Manager and shown in the top menu bar.

Forced Startup Defaults (Entry Point)

  • In fl_atlas.py you can define startup values via FORCE_STARTUP_SETTINGS, STARTUP_LANGUAGE, and STARTUP_THEME.
  • If forcing is enabled, these values overwrite saved user settings on every start.

System & Universe Settings

  • Music, colors, backgrounds, factions, and more can be customized per system.
  • Universe map: Move systems by drag & drop, create new systems.

INI & File Management

  • Automatic backups before every save.
  • INI format remains 100% Freelancer-compatible.
  • In the help window, Reset to factory defaults clears config/cache/history.
