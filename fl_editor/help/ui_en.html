User Interface

Toolbar

  • 🌐 Universe: Open universe map
  • 📈 Trade Routes: Open trade-route view
  • 📝 Name Editor: Search/edit IDS names
  • 🧊 Open Model: 3D model preview
  • Move, Zones, 3D: Checkboxes for map view
  • 🌟 New System: Create new system
  • 💾 Save, ↩ Undo: Save/undo changes
  • 🔍 Missing IDS: Opens Name Editor with missing IDs
  • ⚙ Settings: Global settings (language, theme, BINI, freelancer.ini)
  • Top-right menu area: Shows the currently active editing mod

Panels

  • Welcome: Quick start with introduction and direct navigation to Mod Manager
  • Left: System browser (no path input), INI editor, universe editor, trade-route and name-editor actions
  • Center: 2D/3D map, trade-route page, and name-editor page
  • Right: Quick editor, creation, editing, system settings
  • Status bar loading indicator: A progress bar appears during longer load operations.

Navigation States

  • Tabs that require a configured game/mod context are shown as disabled when unavailable.
  • Mod Manager and Global Settings remain available for setup.

Mod Manager

  • Central place for all mod paths and mod profiles.
  • Right-clicking mod entries opens a context menu (open, edit, activate, deactivate, delete).
  • For direct mods, delete removes only the list entry; the folder stays untouched.
