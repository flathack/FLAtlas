User Interface

Toolbar

  • 🌐 Universe: Open universe map
  • 📈 Trade Routes: Open trade-route view
  • 📝 Name Editor: Search/edit IDS names
  • 🧊 Open Model: 3D model preview
  • Move, Zones, 3D: Checkboxes for map view
  • 🌟 New System: Create new system
  • 💾 Save, ↩ Undo: Save/undo changes
  • 🔍 Missing IDS: Opens Name Editor with missing IDs
  • ⚙ Settings: Global settings including operation mode

Panels

  • Welcome: First-run setup (path, mode, language, theme) for missing/invalid paths
  • Left: System browser, INI editor, universe editor, trade-route and name-editor actions
  • Center: 2D/3D map, trade-route page, and name-editor page
  • Right: Quick editor, creation, editing, system settings
