User Interface

Toolbar

  • 🌐 Universe: Open universe map
  • 🧊 Open Model: 3D model preview
  • Move, Zones, 3D: Checkboxes for map view
  • 🌟 New System: Create new system
  • 💾 Save, ↩ Undo: Save/undo changes
  • 🔍 Missing IDS, 📥 Apply IDS: IDS management
  • 🎨 Theme: Choose color scheme
  • DE/EN: Switch language
  • ℹ️ About: Help & version

Panels

  • Left: System browser, INI editor, universe editor
  • Center: 2D/3D map with objects and zones
  • Right: Quick editor, creation, editing, system settings

Legend & Colors

The color legend at the bottom explains the meaning of symbols (sun, planet, station, jump gate/hole, zone, etc.).
