Overview

FL Atlas is a graphical editor for the game data of Freelancer (2003, Digital Anvil). The program reads INI files (systems, universe, solars, zones, etc.) and displays them on an interactive 2D and 3D map. Standard Freelancer, Freelancer HD Edition, and many mods are supported.

  • Visually edit systems, objects, zones, and connections
  • Complete workflow for bases, tradelanes, jump gates/holes, planets, suns, asteroid fields, and nebulae
  • Live preview and instant feedback on changes
  • Multilingual interface (German/English) and theme selection
  • Compatible with vanilla and mod setups
