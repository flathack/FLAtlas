Overview

FL Atlas is a graphical editor for Freelancer INI data with 2D/3D views and mod-safe write workflows.

  • Visual editing for systems, objects, zones, and connections
  • Mod Manager as the central place for mod paths, activate/deactivate, and editing context
  • Trade Routes as a dedicated main page with multi-system 2D preview
  • Name Editor as a dedicated main page for DLL names and direct ids_name assignment
  • In-game name display via DLL resolution
  • Compatible with vanilla and mod overlay setups
