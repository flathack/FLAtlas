Name-Editor

Im Name-Editor verwaltest du ids_name-Einträge zentral direkt in den Resource-DLLs (User-DLL), ohne CSV-Umweg.

Funktionen

  • Suchen: Nach Global ID, Name oder DLL filtern.
  • Bearbeiten: Vorhandene Namen auswählen und direkt speichern.
  • Neu anlegen: Neue Name-Einträge erzeugen (neue Global ID).
  • Verwendung anzeigen: Bei Klick auf eine ID siehst du alle Objekte/Zonen und Dateien, die diese ID nutzen.

Fehlende ids_name direkt vergeben

  • Der untere Bereich listet alle Objekte/Zonen mit ids_name = 0.
  • Namen eingeben und direkt zuweisen: FL Atlas erstellt den DLL-Eintrag und schreibt die neue ID in die INI-Datei.
  • Keine CSV-Dateien nötig.

Wichtig

  • Änderungen gehen in die konfigurierte User-DLL, nicht in Vanilla-DLLs.
  • Bestehende IDs können wiederverwendet und angepasst werden.
