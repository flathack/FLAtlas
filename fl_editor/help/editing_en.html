Editing & Creation

Objects & Zones

  • Objects and zones can be selected and edited directly.
  • Position, rotation, archetype, faction, loadout, and additional fields are editable.
  • Object/zone dialogs support in-game names; FL Atlas creates/updates the corresponding ids_name entry in the user DLL.

Complex Workflows

  • Trade Routes: Central commodity-route management including 2D route visualization and conflict checks (existing links / duplicate buy targets).
  • Name Editor: Search, edit, create IDS names and assign missing ids_name directly.
  • NPC / Rumor / News Editors: Available as main navigation pages (not popup windows).
  • Bases / Tradelanes / Jumps: Full creation and editing workflows.

Base Creation & Editing

  • Base creation supports room templates, optional NPC copy, and room-level NPC role/reputation/affiliation editing.
  • When template rooms use virtual dealer targets, FL Atlas keeps required virtual-room navigation hotspots.
  • Room NPC fixtures are written in vanilla-like structure to avoid duplicate ambient/vendor spawns.

Undo & Save

  • All changes can be undone (Undo).
  • Backups are created before saving.
