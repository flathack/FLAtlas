Editing & Creation

Objects & Zones

  • Objects and zones can be selected and edited directly.
  • Position, rotation, archetype, faction, loadout, and additional fields are editable.
  • Object/zone dialogs support in-game names; FL Atlas creates/updates the corresponding ids_name entry in the user DLL.

Complex Workflows

  • Trade Routes: Central commodity-route management including 2D route visualization.
  • Name Editor: Search, edit, create IDS names and assign missing ids_name directly.
  • Bases / Tradelanes / Jumps: Full creation and editing workflows.

Undo & Save

  • All changes can be undone (Undo).
  • Backups are created before saving.
