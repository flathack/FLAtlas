FAQ & Troubleshooting

Frequently Asked Questions

  • No system shown? Check game path (DATA + EXE).
  • 3D view is empty? Verify Qt3D installation.
  • Model not found? Check path and case sensitivity.
  • Saving fails? Check write permissions.
  • Assign missing ids_name? Open Name Editor (via 📝 or 🔍 Missing IDS) and assign directly in the lower table.
  • Which objects use an ID? Click an ID row in Name Editor; usage is listed below.
