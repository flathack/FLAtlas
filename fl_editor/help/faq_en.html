FAQ & Troubleshooting

Frequently Asked Questions

  • No system is shown? Check if the game path is set correctly (must contain DATA and EXE).
  • 3D view is empty? Make sure Qt3D is installed (Linux: python3-pyside6.qt3d*).
  • Model not found? Check case sensitivity and path in the game directory.
  • Saving fails? Check write permissions in the game directory.
  • CSV files missing? Run IDS scan first, then edit and import the CSV.
  • How do I change the language? Use the DE/EN button in the toolbar.
  • How do I use a custom theme? Use 🎨 Theme → Custom → select accent color.
